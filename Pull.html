Pulling from a remote repository

If you are working on a different computer, or someone else has updated the remote repository, you need to be able to retrieve the latest commits that are available

To update your local repository to the latest commit, in your Working Directory use the command

git pull

this fetches all remote changes and merges them into your Directory.

Set a user

Remote Repository

Local Repository

Staging

git config user.name "YOUR NAME"

git config user.email "YOUR EMAIL"

Using these will add the user who will make commits to the repository.

You will need to set the username and email that will be used to commit changes

git config user.name "YOUR NAME" git config user.email "YOUR EMAIL"

When you have used these commands, your name and email address will be added in the config file to be used for committing changes in this repo

Creating a Repository

Remote Repository

Local Repository

Staging

git init

This code initiates a local repository

git remote add origin REPOSITORY_URL

This will link your local repository to your remote repository.

Your repository url can be found on the main page of the recently created repository. You can create a repository at bitbucket.org

Once installed, you will want to create a brand new repository, to do this use the command in the commandline of the folder that will contain all of the files in your project

git init

This will create a new local repository on your system

Your next stage is to link your local repository to your remote repository

git remote add origin REPOSITORY_URL

You can create a remote repository at Bitbucket.org or Github.com

If you have a repository that has already been pushed to, you need to checkout a repository

git clone link/to/repository

If you are using a remote repository, the command will be

git clone username@host:link/to/repository
